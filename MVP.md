# Minimum Viable Product (MVP)

## Goal

Develop a basic version of the Internship Management Platform with essential functionality.

---

## Core Features Included

### Authentication Module

* Student Login
* Recruiter Login
* Admin Login

### Student Module

* Browse Internship Listings
* Apply for Internships
* Track Application Status

### Recruiter Module

* Create Internship Postings
* View Applications
* Shortlist Candidates

### Admin Module

* Manage Users
* Monitor Internship Listings

---

## Features Excluded from MVP

* AI Resume Screening
* Video Interview Integration
* Email Automation
* Recommendation Engine
* Real-Time Chat

---

## Success Criteria

* Students can apply for internships.
* Recruiters can manage applications.
* Admins can monitor platform activities.
* Basic internship workflow is functional.
