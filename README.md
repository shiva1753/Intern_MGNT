## Problem Statement

Design an Internship Management Platform that streamlines the internship process for students, recruiters, and administrators. The platform enables students to discover and apply for internships, recruiters to manage applications, and administrators to oversee the entire recruitment workflow.

---

## Objective

The objective of the platform is to simplify internship management by providing a centralized system for posting opportunities, applying to internships, tracking applications, and managing recruitment activities.

---

## User Roles

### Student

* Register and Login
* Browse Internships
* Apply for Internships
* Track Application Status

### Recruiter

* Create Internship Postings
* Review Applications
* Shortlist Candidates
* Update Application Status

### Admin

* Manage Users
* Approve Internship Listings
* Generate Reports
* Monitor Platform Activity
